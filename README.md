# 🚀 AWZION Finance v2

Sistema financeiro profissional desenvolvido em PHP, MySQL e CSS moderno.

---

# 📌 Sobre o Projeto

O **AWZION Finance v2** é um sistema criado para controle financeiro pessoal e empresarial.

O sistema permite:

✅ Controle de ganhos  
✅ Controle de gastos  
✅ Categorias financeiras  
✅ Dashboard moderno  
✅ Relatórios mensais  
✅ Sistema de login  
✅ Sessões protegidas  
✅ Interface responsiva  
✅ Gráficos financeiros  

---

# 🖥️ Tecnologias Utilizadas

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- Chart.js

---

# 🔐 Segurança

O arquivo `config.php` não está incluído neste repositório por segurança.

Crie um arquivo:

```php
config.php
```

Baseado em:

```php
config.example.php
```

---

# ⚙️ Configuração do Banco

Configure:

```php
$dbHost
$dbName
$dbUser
$dbPass
```
